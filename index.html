CONFIG FIREBASE

Pour que le bingo soit partagé entre plusieurs PC en temps réel,
tu dois configurer Firebase (gratuit, ~5 min).

1. Va sur console.firebase.google.com
2. Crée un projet → Realtime Database → mode test
3. Projet > Paramètres > Tes applications → copie ta config
4. Ouvre ce fichier HTML dans un éditeur et remplace les REMPLACE_MOI dans la section firebaseConfig

const firebaseConfig = { apiKey: "...", authDomain: "...", databaseURL: "...", projectId: "...", storageBucket: "...", messagingSenderId: "...", appId: "..." };

Une fois fait, recharge la page.
